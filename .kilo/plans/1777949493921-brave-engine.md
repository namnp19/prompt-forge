# Gợi ý tên cho Extension VS Code "Overwrite"

## Mô tả Extension

Extension này có 3 chức năng chính:
1. **Explorer** – Duyệt và chọn file/folder từ workspace
2. **Context** – Đóng gói nội dung file thành XML prompt gửi cho LLM
3. **Apply** – Nhận XML response từ LLM và áp dụng thay đổi trở lại file local

---

## Các nhóm tên đề xuất

### Nhóm 1: Nhấn mạnh việc "đóng gói code → AI"
| Tên | Lý do |
|---|---|
| **PromptForge** | "Rèn" prompt từ code của bạn |
| **PromptPack** | Đóng gói file thành prompt |
| **RepoPrompt** | Chuyển repo thành prompt (đơn giản, trực tiếp) |
| **ContextCraft** | Craft/tạo context cho AI |
| **CodeCast** | "Cast" (truyền) code tới AI |

### Nhóm 2: Nhấn mạnh vòng lặp "code ↔ AI"
| Tên | Lý do |
|---|---|
| **Shuttle** | Con thoi đưa code đến AI và ngược lại |
| **CodeRelay** | Relay (chuyển tiếp) code tới AI |
| **Conduit** | Ống dẫn giữa code và AI |
| **CodeBridge** | Cầu nối giữa code và AI |
| **Nexus** | Điểm kết nối giữa codebase và AI |

### Nhóm 3: Nhấn mạnh việc "áp dụng thay đổi"
| Tên | Lý do |
|---|---|
| **PatchForge** | Tạo và áp dụng patch từ AI |
| **Graft** | Ghép thay đổi từ AI vào code |
| **Stitch** | "Khâu" các thay đổi AI vào codebase |
| **Splice** | Ghép nối code với đề xuất AI |
| **Weave** | Dệt các thay đổi AI vào code |

### Nhóm 4: Tên ngắn gọn, dễ nhớ
| Tên | Lý do |
|---|---|
| **Loom** | Khung cửi – dệt code thành prompt và ngược lại |
| **Flux** | Dòng chảy liên tục giữa code và AI |
| **Loop** | Vòng lặp AI-coding |
| **Relay** | Chuyển tiếp nhanh |

---

## Top 5 đề xuất cá nhân

1. **PromptForge** ⭐ – Mạnh mẽ, chuyên nghiệp, dễ nhớ; truyền tải rõ chức năng tạo prompt
2. **Shuttle** ⭐ – Ngắn gọn, hình ảnh sinh động (con thoi code ↔ AI)
3. **ContextCraft** ⭐ – Rõ ràng về chức năng, phù hợp thị trường developer
4. **RepoPrompt** ⭐ – Cực kỳ trực tiếp, dễ tìm kiếm trên Marketplace
5. **Loom** ⭐ – Ngắn, độc đáo, ẩn dụ đẹp (dệt code → AI → patch)
